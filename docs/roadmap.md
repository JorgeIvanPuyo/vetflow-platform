# Roadmap activo

Este documento es la única fuente de estado y prioridades activas del MVP. Para la secuencia ya completada, consulta `archive/mvp-roadmap-history.md`.

## Estado actual

Vetflow ya cubre el flujo clínico central multi-tenant: autenticación, propietarios, pacientes, historia clínica, consultas, exámenes, cuidados preventivos, archivos clínicos, agenda, seguimientos, dashboard, perfil de clínica e inventario MVP.

## Prioridades activas

1. **Plan terapéutico con inventario:** selector de medicamentos, validación de stock, descuento visible y mensajes de insuficiencia.
2. **Prescripciones:** generación, exportación y relación con la consulta.
3. **Recordatorios:** seguimientos, vacunas y turnos.
4. **Administración de clínica:** invitaciones, roles y endurecimiento de configuración.
5. **Estabilización para pilotos:** UX, casos borde, pruebas, accesibilidad, observabilidad y checklist operativo.

## Fuera del MVP actual

- Aplicación móvil nativa.
- Analítica avanzada.
- Valuación avanzada de inventario, órdenes de compra y código de barras.
- Notificaciones por WhatsApp e integración con Google Calendar.

## Uso

Actualiza este documento cuando cambie la prioridad, el estado de una iniciativa o la definición del MVP. Las decisiones detalladas viven en el PRD o backlog correspondiente.
